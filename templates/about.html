About Our Project

Smart Code Assistant is a cutting-edge AI-powered tool developed as our final year Computer Science project. The goal is to help developers write cleaner, more efficient, and error-free code using the latest AI technologies.

Key Features

  • Find and fix bugs in any code snippet using natural language input
  • Optimize code for better time and space complexity
  • User-friendly web interface with prompt-based AI interactions

Technology Stack

  • Model: llama3.2:7b via Ollama server
  • Backend: Python Flask
  • Frontend: HTML, CSS, JavaScript
  • CI/CD: GitHub Actions
  • Deployment: Dockerized application pushed to Docker Hub

Our Mission

We aim to reduce the complexity of debugging and optimizing code by offering an intelligent assistant that can understand code and developer intent, making software development faster and more reliable.
