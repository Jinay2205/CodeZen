About Our Project

CodeZen is a smart code analysis tool designed to improve code quality and performance. It assists users by detecting bugs or optimizing code using advanced language models. Built for ease of use, it ensures fast, reliable, and secure code feedback.

Key Features

  • Find and fix Bugs
  • Optimize code
  • User-friendly interface

Technology Stack

  • Model: llama3.2:7b via Ollama server
  • Backend: Python Flask
  • Frontend: HTML, CSS, JavaScript

Our Mission

To simplify the debugging process and make code improvement effortless for everyone.
